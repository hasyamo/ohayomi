# おはヨミ

お気に入りのnoteクリエイターの朝活記事を、朝のうちにチェック＆コメントする習慣をつくるPWA

https://hasyamo.github.io/ohayomi/

## 機能

- **クリエイター登録** - noteのURLを貼り付けるだけで自動登録
- **チェックリスト** - 読んだ / コメントした をタップで記録
- **新着判定** - 記事数の変化で NEW バッジを表示
- **日次リセット** - 毎朝5時にステータスを自動リセット
- **並び替え** - 設定画面でクリエイターの表示順を変更
- **アーカイブ** - 一時的に非表示にして後から復帰可能
- **エクスポート / インポート** - JSONファイルでバックアップ・復元
- **PWA対応** - ホーム画面に追加してアプリとして利用

## 技術構成

- Vanilla JS + Vite
- localStorage
- Cloudflare Workers（APIプロキシ）
- GitHub Pages

## 開発

```bash
npm install
npx vite
```
